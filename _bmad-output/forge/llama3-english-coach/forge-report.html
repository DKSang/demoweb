Locked Specifications
YouTube Shadowing Player
Embeds the YouTube IFrame API to play, pause, and loop specific video lessons from selected playlists. Features dynamic playback speed control for beginner listening comfort.
Pronunciation Assessor
Utilizes Chrome/Edge's native Web Speech API (`en-GB`/`en-US`) to transcribe speech, performing client-side Levenshtein text-matching against target subtitles to highlight mistakes in real time.
Vocabulary dynamic injection
Locks a target of 25 popular words per day. The active words are dynamically injected into Llama-3-8b’s prompt templates during conversation sessions to encourage contextual usage.
Ollama client proxy
Connects directly to your local Llama-3-8b instance through a Vite proxy (`/api/ollama` → `127.0.0.1:11434`), enforcing environmental rules without exposing localhost strings.
Client-only Local Database
Saves all vocabulary notebook entries, streak calendars, and conversational transcripts directly in Chrome `localStorage`, removing complex database dependencies.
Discarded / Changed Ideas
  • Complex Grammar Analysis: Removed heavy, academic grammar breakdowns to prevent cognitive fatigue. Llama-3-8b will output single-sentence, actionable correction cards instead.
  • Server-Side Storage: Postponed Express/database layers. Sticking strictly to a zero-install browser storage flow to ensure maximum local speed and simple code footprint.
  • 5-Word Daily Limit: Standard 5-word limits killed. Upgraded to a target-rich 25-word list daily to hit popular word benchmarks.
